## Migration Notes

### LLM provider 활성화 (옵션 — default disabled)
```bash
# Ollama (CPU OK, edge 적합):
export ROSSHIELD_LLM_PROVIDER=ollama
export ROSSHIELD_LLM_BASE_URL=http://127.0.0.1:11434
export ROSSHIELD_LLM_MODEL=llama3.1:8b
export ROSSHIELD_LLM_AUTO_PULL=true  # 모델 자동 download
ollama pull llama3.1:8b

# vLLM (GPU 필수, data center 적합):
export ROSSHIELD_LLM_PROVIDER=vllm
export ROSSHIELD_LLM_BASE_URL=http://vllm.internal:8000
export ROSSHIELD_LLM_MODEL=meta-llama/Llama-3.1-8B-Instruct
export ROSSHIELD_LLM_MAX_TOKENS=2048
docker run --gpus all -p 8000:8000 vllm/vllm-openai --model meta-llama/Llama-3.1-8B-Instruct
```

상세는 `docs/onboarding/llm-private-deployment.md` 참조.

### 빌드
```bash
git fetch --tags && git checkout v0.6.4
cd web && pnpm install --frozen-lockfile && pnpm build
cd .. && go build -o bin/rosshield-server ./cmd/rosshield-server
./bin/rosshield-server --addr 127.0.0.1:8080 --data-dir ./run-data
```
