## Migration Notes

- **DB 마이그레이션 0건**.
- **enterprise build**: `make build-enterprise` 또는 `go build -tags rosshield_enterprise ./...` 그대로 동작. v2 보강 자동 포함.
- **신규 dep 0** — `go.mod`/`go.sum` 변경 0 (stdlib + 이미 indirect dep `go-tpm-tools`, `lukechampine.com/blake3`, `tetratelabs/wazero` 활용).
- **R-D8 v2 사용 예** (옵션):
  - `crosswitness.NewScheduler(...)` — interval fold-in 자동화 (기본 비활성, 사용자가 명시 활성화)
  - `multihash.Compute(... Option{JSONPaths: []string{"$.checks[*].status"}})` — wildcard로 100 check sub-hash 자동 expand
  - `wasmrt.CosignKeyedVerifier{PublicKey: ..., HashFn: crypto.SHA256}` — WASM policy 서명 검증
  - `robotid.Compute(Inputs{..., PCRValues: ...})` — PCR 결합 fingerprint (TPM 있는 환경 한정)
