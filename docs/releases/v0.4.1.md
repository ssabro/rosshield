## Migration Notes

- **DB 마이그레이션**: 0024 evidence_json JSONB cast DEFAULT DROP/SET 패턴으로 fix됨. v0.4.0 → v0.4.1 업그레이드 시 마이그레이션 재실행 시 자동 적용. 신규 설치는 영향 0.
- **Secret 등록 권장**: CI 빌드를 위해 GitHub repo Secret `DEV_PACK_SIGNER_KEY_B64`(base64 `scripts/dev-pack-signer.key`) 등록 필요. 외부 contributor 빌드는 fallback keygen으로 vet/build PASS 보장 (단 InstallPack Verify는 mismatch).
- **snapcraft 8.x 환경**: snap binary 빌드 시 `architectures: [{build-on: amd64, build-for: amd64}, {build-on: amd64, build-for: arm64}]` 사용 — arm64 native runner 의존 없음.
