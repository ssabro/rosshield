## Migration Notes

- **빌드**: `pnpm install` (5 신규 dep) → `pnpm build` → Go server binary 재빌드 `make build` 또는 `go build ./cmd/rosshield-server`
- **PWA 캐시**: 기존 사용자 브라우저 강제 새로고침 (Ctrl+Shift+R) 권장 (sw.js 갱신 + virtual:pwa-register chunk 신규)
- **PWA install 사용자**: 앱 이름 "rosshield Console" → "Lodestar 관리자 콘솔" 자동 갱신
- **Locale**: 한국어 default 유지 (navigator.language 추정), Header LocaleToggle로 EN 전환 가능
- **Theme**: light/dark/system 3 mode, localStorage persist
