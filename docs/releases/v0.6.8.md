## Added

### cosign keyless Sigstore 실서명 (`904c55e`)

design doc D-AR-4 — audit rotation archive에 cosign keyless 서명 + bundle 저장.

#### Signer interface
```go
type Signer interface {
  Sign(ctx context.Context, archive []byte) ([]byte, error)
  Enabled() bool
}
```

#### CosignSigner — 옵션 A 외부 cosign CLI
- `cosign sign-blob --yes --bundle=- [--fulcio-url=...] [--rekor-url=...] -` (stdin=archive, stdout=bundle)
- cosign binary가 PATH에 있거나 `--cosign-binary` 지정 필요
- FakeSigner — test 용 (signFunc nil이면 비활성)

#### rotation.go 결선
- `Archive()` 호출 직후, `insertSegment()` 직전
- `signArchive(ctx, uri)` → `backend.Get(uri)` 후 `signer.Sign(ctx, body)` → bundle을 `SegmentRecord.CosignBundle`에 저장
- 마이그레이션 0032의 reserve 컬럼 재사용 (신규 마이그레이션 0)
- signer nil/Enabled=false → bundle=nil → DB NULL

#### Bootstrap env 5건
- `ROSSHIELD_COSIGN_ENABLED` · `_BINARY` · `_IDENTITY` · `_FULCIO_URL` · `_REKOR_URL`
- CLI flag 5건 (priority flag > env)

#### 운영 doc
`docs/onboarding/audit-rotation-cosign.md` (155줄) — keyless 의미 + 설치 + OIDC token 3 방식 + verify 절차 + 한계.

#### Test (14)

design doc Stage 3 — primary `CREATE PUBLICATION` + standby `CREATE SUBSCRIPTION` 자동 idempotent.

## Migration Notes

### cosign 활성화 (옵션)
```bash
# cosign binary 설치
curl -LO https://github.com/sigstore/cosign/releases/latest/download/cosign-linux-amd64
chmod +x cosign-linux-amd64 && sudo mv cosign-linux-amd64 /usr/local/bin/cosign

export ROSSHIELD_COSIGN_ENABLED=true
export ROSSHIELD_COSIGN_IDENTITY=ci@example.com
```

### HA Stage 3 자동 setup (옵션)
```bash
# Primary
export ROSSHIELD_REPLICATION_AUTO_SETUP=true
export ROSSHIELD_REPLICATION_ROLE=primary

# Standby
export ROSSHIELD_REPLICATION_AUTO_SETUP=true
export ROSSHIELD_REPLICATION_ROLE=standby
export ROSSHIELD_REPLICATION_PRIMARY_CONN_STRING='host=primary.us-west-2 user=replica dbname=rosshield'
```

PG 설정 필수: `wal_level = logical`, `max_replication_slots`, `max_wal_senders`.

### S3 backend (enterprise 빌드)
```bash
go build -tags rosshield_enterprise -o bin/rosshield-server ./cmd/rosshield-server

export ROSSHIELD_AUDIT_COLD_BACKEND=s3
export ROSSHIELD_AUDIT_S3_BUCKET=my-audit-bucket
export ROSSHIELD_AUDIT_S3_REGION=us-west-2
export ROSSHIELD_AUDIT_S3_PREFIX=audit-archives/
export ROSSHIELD_AUDIT_S3_SSE=AES256
```

### 빌드
```bash
git fetch --tags && git checkout v0.6.8
cd web && pnpm install --frozen-lockfile && pnpm build
cd .. && go build -o bin/rosshield-server ./cmd/rosshield-server
# enterprise (S3 활성):
go build -tags rosshield_enterprise -o bin/rosshield-server ./cmd/rosshield-server
```
