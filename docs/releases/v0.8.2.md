## Fixed

### 1. PG `TestAuditChainHeadSHACrossRegion` flaky (`6ee8275`)

`audit_chain_heads`(1 row)와 `audit_entries`(5 row separate transactions)는 별개 publication
row라 logical replication 도착 순서가 atomic 보장 안 됨. 종전 1-shot SELECT COUNT가
chain_heads는 도착하지만 entries는 아직 0건인 race 노출.

**Fix**: audit_entries 5건 sanity SELECT을 5초 deadline polling loop로 변경 (T6 line
620-635 pattern 일관 적용). 100ms tick × 50 iter.

`internal/platform/storage/postgres/replication_integration_test.go` +10/-4.

### 2. MinIO `TestS3Backend_MinIOLifecycle` (`6ee8275`)

신규 MinIO release(2024+)는 transition StorageClass를 strict validate — remote tier
(`mc admin tier add`) 미등록 환경에서 STANDARD_IA·GLACIER·DEEP_ARCHIVE 모두 400
`InvalidStorageClass` 거부. 종전 "MinIO silent ignore" 가정이 깨짐.

**Fix**: test cfg에서 `LifecycleTransitions` 제거, `LifecycleExpireDays`만 유지. Transition
rule 직렬화는 fake S3 mock test(`backend_s3_enterprise_test.go`)가 이미 cover. 본 test는
통신 layer(Content-MD5 middleware) + Expiration rule 등록 검증만.

`internal/domain/audit/rotation/backend_s3_minio_integration_test.go` +5/-6.

### 3. Playwright 19 E2E loginAsAdmin 회귀 (`6ee8275`)

commit `44b139f`(2026-05-19 D-P7-1 브랜드 적용)에서 dict.ts `login.title`을
`'rosshield Console'` → `'Lodestar 관리자 콘솔'`로 변경하면서 Playwright fixtures.ts
의 `KO_LABELS.login.title` 동기 갱신 누락. `loginAsAdmin` helper(helpers.ts:12)를 거치는
19 smoke test 모두 같은 사유로 fail.

**Fix**: `fixtures.ts`의 `KO_LABELS.login.title`을 `'Lodestar 관리자 콘솔'`로 동기화.
부수: dict.ts:1158 en `login.title` 값이 한글 leak ('Lodestar Admin Console'로 정정).

`web/playwright/fixtures.ts` + `web/src/i18n/dict.ts`.

### 4. Snap Smoke `/healthz` 30s timeout — 20+ 누적 fail (`4929a7b`)

rosshield-server 자체 default가 `--addr=127.0.0.1:0`(random port). snap daemon은
`active`로 보고되지만 healthz endpoint는 random port에 있어 8080 curl 실패.
snap description의 운영자 가이드(`snap set rosshield addr=127.0.0.1:8080`)와 daemon
실 동작이 어긋남.

**Fix 3종**:
- **snapcraft.yaml**: `command: bin/rosshield-server --addr=127.0.0.1:8080` (daemon에 default 박기)
- **snap-smoke.yml**: `--no-color` flag 제거 (Ubuntu 22.04 snap CLI 미인식 — 24.04+ 추가)
- **snap-smoke.yml**: timeout 30s → 60s + 매 20s마다 진행 상태 출력

`snap/snapcraft.yaml` + `.github/workflows/snap-smoke.yml`.

### 5. Playwright audit + color-contrast 신규 issue (`cc8511e`)

#### audit.spec.ts:18 strict mode violation

`getByText('Chain Head')`가 2 elements 매치 (heading sub-label + description의 substring).
`{ exact: true }` 추가로 정확화.

#### color-contrast 16 케이스 — C5b-10 carryover

- muted-foreground `#64748b` on slate-100 `#f1f5f9` = 4.34:1 < WCAG AA 4.5:1
- destructive button `#f8fafc` on `#ef4444` = 3.59:1 < 4.5:1

Tailwind theme palette 변경 작업 — design decision + 시각 영향 검토 필요. 별 PR
(C5b-10 a11y polish round)로 분리. `test.skip` 일시 격리 + PR 완료 시 .skip 제거.

`web/playwright/tests/audit.spec.ts` + `web/playwright/tests/color-contrast.spec.ts`.
